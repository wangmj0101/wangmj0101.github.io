    Hexo博客搭建

    环境准备

    node.js

    下载地址：https://nodejs.org/dist/

    最新hexo版本，nodejs建议下载>=18.0

    查看安装的版本

    node -v
npm -v

    替换镜像源

    npm install -g cnpm --registry=https://registry.taobao.org
# 查看安装
cnpm -v

    安装Hexo

    cnpm安装hexo

    使用cnpm全局安装hexo

    cnpm install -g hexo-cli

    安装后验证安装

    hexo -v

    创建文件夹并初始化

    进入创建的文件夹，在此目录下进行hexo初始化

    hexo init

    初始化后该目录下的文件结构

    HexoBlog
    |- _config.landscape.yml
    |- _config.yml   #配置文件
    |- db.json
    |- node_modules/
    |- package.json
    |- public/       #公共静态资源问价夹
    |- scaffolds/
    |- source/       #资源文件(写作md文件)
        |-- _post/
            |-- HelloHexo.md
            |-- 1.md
    |- themes/       #主题文件夹
        |-- 3-hexo/  #安装的第三方主题
    |- yarn.lock

    启动Hexo

    启动命令中s表示start

    hexo s

    新建文章

    新建文章命令

    hexo n "测试文章"

    n表示new，该命令会在source/_post/目录下创建出测试文章.md
